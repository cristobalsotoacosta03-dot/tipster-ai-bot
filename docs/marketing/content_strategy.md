## 📊 TikTok/Reels/Shorts Strategy

### Fórmula de Contenido Viral

**Estructura del video (30-60s):**
```
0-3s: HOOK (gancho visual/emocional)
3-15s: CONTENIDO (análisis/value)
15-30s: PRUEBA SOCIAL (resultados/estadísticas)
30-45s: CTA (llamada a la acción)
45-60s: CALLBACK (cierre memorable)
```

### Tipos de Contenido

#### 1. **Análisis Express** (40% del contenido)
**Formato:** "Este partido tiene VALUE"
- Muestra el análisis del bot
- Destaca métricas clave (xG, PPDA)
- CTA: "Link en bio para análisis completo"

**Ejemplo guión:**
```
[0-3s] "El Real Madrid vs Barcelona tiene VALUE oculto"
[3-15s] "El Madrid genera 1.8 xG en casa, el Barcelona solo 1.2 de visita"
[15-30s] "Además, el Madrid tiene 2 días más de descanso"
[30-45s] "Análisis completo en mi Telegram, link en bio"
[45-60s] "No apuestes sin datos"
```

#### 2. **Educación Apuestas** (30% del contenido)
**Formato:** "Deja de apostar así"
- Errores comunes
- Conceptos básicos (value betting, stake management)
- Posicionamiento como experto

**Ejemplo guión:**
```
[0-3s] "Deja de apostar a tu equipo favorito"
[3-15s] "El 90% de apostadores pierden por sesgo emocional"
[15-30s] "Yo uso IA para analizar pressing, fatiga, xG..."
[30-45s] "[PENDIENTE: sustituir por cifra de acierto real cuando haya histórico suficiente — no publicar con un dato inventado]"
[45-60s] "Link en bio para análisis objetivos"
```

#### 3. **Resultados/Proof** (20% del contenido)
**Formato:** "Acierto del día"
- Muestra pronósticos acertados
- Estadísticas reales
- Credibilidad

**Ejemplo guión:**
```
[0-3s] "[PENDIENTE: usar un resultado real y verificable del bot, no un ejemplo inventado]"
[3-15s] "[PENDIENTE: cifra de confianza real de ese análisis, tal como la reportó el bot]"
[15-30s] "El Atlético presionó más (PPDA 8.5) y ganó"
[30-45s] "Hoy tengo 3 análisis premium, link en bio"
[45-60s] "El value betting es a largo plazo"
```

#### 4. **Detrás de Escenas** (10% del contenido)
**Formato:** "Cómo analizo un partido"
- Proceso de trabajo
- Herramientas (Claude AI, API-Football)
- Transparencia

**Ejemplo guión:**
```
[0-3s] "Así analizo un partido en 20 segundos"
[3-15s] "Consulto 40+ métricas: xG, PPDA, fatiga..."
[15-30s] "Claude AI genera el análisis táctico"
[30-45s] "Te lo envío a Telegram, link en bio"
[45-60s] "La tecnología gana a la intuición"
```

## 💡 Tips para Viralidad

### TikTok Algorithm Hacks

1. **Primeros 3 segundos** - El hook es CRÍTICO
   - Usa números: "El error #1..."
   - Usa preguntas: "¿Por qué pierdes dinero?"
   - Usa sorpresa: "Este partido tiene VALUE oculto"

2. **Texto en pantalla** - Muchos ven sin sonido
   - Subtítulos grandes
   - Keywords destacados
   - CTA visible

3. **Hashtags estratégicos**
   - 3-5 hashtags grandes (#apuestas, #futbol)
   - 3-5 hashtags medianos (#tipster, #apuestasdeportivas)
   - 2-3 hashtags pequeños (#tipsteria, #valuebetting)

4. **Call to Action claro**
   - "Link en bio"
   - "Comenta 'VIP' para más info"
   - "Sígueme para más tips"

5. **Consistencia**
   - 1 video/día mínimo
   - Mejor horario: 19:00-21:00 (España)
   - Responder todos los comentarios
